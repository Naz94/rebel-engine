How to drop a block in
1
Open your page in VS Code (or any editor)
Open harvest-co/index.html or whichever page you're editing.
2
Find where you want the block
Look for the section comments like <!-- ══ FEATURED PRODUCTS ══ -->. Place your cursor above or below it.
3
Copy the block code from this library
Click "Copy Code" on any block card. It copies everything between BLOCK START and BLOCK END — the HTML, CSS, and JS all included.
4
Paste it in, save, refresh
That's it. The block reads your theme colours automatically. No CSS file editing needed.
/* Before */ <!-- ══ FEATURED PRODUCTS ══ --> <section id="section-featuredProducts">... /* After: paste the block above it */ <!-- ═══ BLOCK START: stats-counter ═══ --> <section class="re-stats">... <!-- ═══ BLOCK END: stats-counter ═══ --> <!-- ══ FEATURED PRODUCTS ══ -->
5
Add re-blocks.js once per page
If using ANY animated block (parallax, carousel, stats counter, before/after), add this script just before </body>.
<script src="../blocks/re-blocks.js"></script> <script src="STORE_CONFIG.js"></script> <script src="../shared/rebel-engine.js"></script>

Your Agency Workflow

From "new client signed" to "live store" — exactly what you do at each step.

New Client Onboarding → Live Store

→
1
Create tenant
Log into super-admin panel. Create new tenant. Copy the generated clientId.
→
2
Pick template
Butchery/food? → harvest-co
Gym/supplements? → kinetic
Copy the folder.
→
3
Fill STORE_CONFIG.js
Name, colours, hero text, categories, contact details. One file. 5–10 minutes.
→
4
Add custom blocks
Open pages in editor. Copy blocks from this library. Drop them in where needed. Use AI for anything custom.
5
Deploy + handover
Upload to client domain. Give them their dashboard login. Done — they manage products/orders themselves.
